Servo airboat rudder by dudufreeride on Thingiverse: https://www.thingiverse.com/thing:4788785

Summary:
This is a simple rudder that I have made for a homemade airboat. It can be useful in other projects, it is intended to be used with an SG90 servo.It is intended to use M4 screws. The final version includes several holes to choose the depth of the rudder in the water.Tip: Using a drop of hot glue on the servo arm prevents it from detaching
